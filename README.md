# Simple Portfolio Website

A simple personal portfolio website built using **HTML and CSS only** (no frameworks or JavaScript), showcasing my introduction, skills, and interests.

## 🔹 Pages

- **Home Page** – Includes my name, a brief introduction about what I do, and my interests, along with a link to the Contact page.
- **Contact Page** – Contains a contact form (HTML markup only; not yet connected to a backend or email service, so submissions are not currently functional).

## 🔹 Features

- Clean, minimal design using basic HTML and CSS styling
- Simple navigation between Home and Contact pages

## 🔹 Tech Stack

- HTML5
- CSS3

## 🔹 Status

Beginner-friendly project built to practice core HTML and CSS concepts. Contact form functionality (e.g., email submission via a service like Formspree) is planned for a future update.
